Checkbox based on form control with label from variable

Below you see several ion-checkboxes based on form controls. The first two are enabled, the second two are disabled by setting the form control to disabled. Of each set of two, the first is checked by default, the second is unchecked by default, by providing a default value true or false in the form control. All checkboxes but one get their label from a variable in the component.

Below you can see that the checkbox which is based on an enabled form control, default false, with label from variable, initially does not show a label, but when you check the box, the label appears.

This happens with ionic/angular 8.7.1 or 8.7.2, but not with 8.7.0. It also happens with ionic/angular 8.6.6, but not with 8.6.5 or 8.6.7. It seems related to the upgrade/downgrade of stencil/core, see the Ionic release notes.
